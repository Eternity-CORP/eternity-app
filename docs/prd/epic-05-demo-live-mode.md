## Реализация

### Сервис
`mobile/src/services/networkModeService.ts`
- `getNetworkMode()` — получить текущий режим
- `setNetworkMode(mode)` — установить режим
- `isNetworkAvailable(network, mode)` — проверить доступность сети
- `getAvailableNetworks(mode)` — получить список сетей

### UI компоненты
- `NetworkModeSwitcher.tsx` — модальное окно выбора режима
- Интеграция в `SettingsScreen.tsx`
- Индикатор в `HomeScreen.tsx`

### Логика
- Demo Mode (по умолчанию): Sepolia, Holesky
- Live Mode: только Mainnet
- Режим сохраняется в AsyncStorage
- При смене режима — автоматическая смена сети
