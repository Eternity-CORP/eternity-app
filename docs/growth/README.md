## Quick Start

### Getting First Users (Week 1-4)
1. Set up Twitter/X presence
2. Join Telegram communities (UA + international)
3. Discord showcase posts
4. Direct outreach to 50 contacts

### Fundraising Priority
1. **Grants first** (no equity): Polygon, Base, Ukrainian Startup Fund
2. **Accelerators** (after 50+ users): Alliance DAO, Outlier Ventures
3. **VC/Angels** (after metrics): Pre-seed $100K-500K
