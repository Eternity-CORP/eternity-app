# Twitter Post Templates

> Templates for E-Y Twitter content

---

## Post Types

### 1. Build in Public

```
Day [X] building E-Y:

[What you did today]

[Screenshot/video if available]

[Question or call to action]
```

**Example:**
```
Day 12 building E-Y:

Added BLIK code expiration timer.
Now you see exactly how much time left to enter the code.

What else would make crypto payments feel safer?
```

## Best Practices

1. **Hook first** - First line must grab attention
2. **One idea per post** - Don't overload
3. **Visual content** - Screenshots/videos increase engagement 3x
4. **Ask questions** - Drives comments
5. **Post timing** - 9-11 AM and 2-4 PM UTC for crypto Twitter
6. **Hashtags** - Use sparingly: #buildinpublic #crypto #web3
