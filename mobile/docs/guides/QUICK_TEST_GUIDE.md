# 🚀 Quick Test Guide - Отправка ETH

## ✅ Готово к тестированию!

Новый `SendEthScreen` уже подключен к навигации и готов к использованию.

## 📱 Как протестировать в приложении

### 1. Запустить приложение

```bash
cd mobile
npm start
```

Выбери платформу:
- `i` - iOS Simulator
- `a` - Android Emulator
- Или отсканируй QR код на реальном устройстве

### 2. Открыть экран отправки

В приложении:
1. На главном экране нажми кнопку **"Send"** или **"Отправить"**
2. Откроется новый `SendEthScreen` с улучшенным интерфейсом

### 3. Отправить тестовую транзакцию

#### Шаг 1: Ввести адрес получателя
```
Можешь использовать любой валидный адрес, например:
0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb
```

- ✅ Валидный адрес - зеленая рамка
- ❌ Невалидный адрес - красная рамка + ошибка

#### Шаг 2: Ввести сумму
```
Например: 0.001
```

- Проверь что у тебя достаточно баланса
- Учитывается комиссия газа (~0.00015 ETH)

#### Шаг 3: Выбрать уровень комиссии

Три варианта:
- **Low** - медленнее, дешевле (~30-60 сек)
- **Medium** - баланс (рекомендуется) (~15-30 сек)
- **High** - быстрее, дороже (~10-20 сек)

#### Шаг 4: (Опционально) Настроить газ вручную

Нажми **"Show Advanced Options"**:
- Gas Limit (обычно 21000 для ETH)
- Max Fee (в Gwei)
- Priority Fee (в Gwei)

#### Шаг 5: Отправить

1. Свайпни **"Slide to send"** вправо
2. Дождись статуса "Sent"
3. Наблюдай за подтверждениями (0 → 1 → 2)
4. Получи финальный alert с деталями

### 4. Проверить транзакцию

#### В приложении:
- Hash отображается в статус-карточке
- Количество подтверждений обновляется в реальном времени
- Финальный alert показывает block number и gas used

#### В Etherscan:
```
https://sepolia.etherscan.io/tx/[ВАШ_HASH]
```

Замени `[ВАШ_HASH]` на hash из приложения.

## 🐛 Возможные проблемы

### "RPC Error" или "Network Error"
**Решение:**
1. Проверь интернет-соединение
2. Убедись что Alchemy API key настроен в `.env`
3. См. `RPC_CONFIGURATION.md`

### "Insufficient funds"
**Решение:**
1. Получи tETH из faucet: https://sepoliafaucet.com/
2. Проверь баланс на главном экране
3. Учитывай комиссию газа

### "Transaction timeout"
**Решение:**
1. Используй 'high' fee level
2. Проверь статус сети (congestion)
3. Увеличь timeout в коде (если нужно)

### Транзакция не подтверждается
**Решение:**
1. Проверь в Etherscan - может быть pending
2. Подожди дольше (иногда сеть медленная)
3. Попробуй с 'high' fee level
