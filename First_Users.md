> Цель: 50-100 бета-тестеров за 2-3 недели

## 1. Twitter/X — главная площадка для крипты

### Почему важно
- Crypto Twitter (CT) — где все: разработчики, инвесторы, пользователи
- Вирусность выше чем где-либо
- Инвесторы смотрят на Twitter presence

### Оформление профиля
```
Имя: E-Y
Handle: @ey_wallet (или похожий)
Bio: "Building E-Y — send crypto with 6-digit code. Like BLIK for crypto. 🔐"
Link: landing page
```

### Контент-план (3-5 постов в неделю)
- **Build in public:** "Today I added X feature"
- **Проблема/решение:** "Sending crypto is broken. Here's why..."
- **Демо видео:** 15-30 сек как работает BLIK
- **Мемы:** про крипто UX

### Аккаунты для follow и engagement
| Аккаунт | Категория |
|---------|-----------|
| @0xPolygon | L2 сеть |
| @base | L2 сеть |
| @arbitrum | L2 сеть |
| @optimism | L2 сеть |
| @AlchemyPlatform | Инфраструктура |
| @WalletConnect | Wallet space |
| @VitalikButerin | Ethereum |

### Пример первого поста
```
I'm building E-Y — a crypto wallet where you send money with just 6 digits.

No addresses. No network selection. No gas confusion.

Like BLIK, but for crypto.

Looking for 50 beta testers. Android ready, iOS next week.

Who wants to try? 👇
```

## 4. Прямой outreach — самый эффективный

### Где искать людей лично
- Друзья кто в крипте
- Коллеги/бывшие коллеги
- Twitter followers кто активен в крипте
- Участники хакатонов (ETHGlobal, etc.)
- LinkedIn контакты из tech

### Шаблон личного сообщения
```
Привет!

Делаю крипто-кошелёк, ищу честный фидбек.
Можешь 5 минут потестить на Android?

Интересно твоё мнение как человека в теме.
Просто напиши да/нет, без обязательств 🙂
```

### Цель: 50 личных сообщений → 15-20 тестеров

## 5. Product Hunt — для launch day

### Когда использовать
- НЕ сейчас, а когда есть 100+ пользователей
- Один день может дать 500-1000 signups
- Нужна подготовка: скриншоты, видео, описание

### Что подготовить заранее
- [ ] 5-6 скриншотов приложения
- [ ] Демо видео 30-60 сек
- [ ] Tagline: "Send crypto with 6 digits — like BLIK for crypto"
- [ ] Описание 2-3 параграфа
- [ ] Отзывы от бета-тестеров

## 8. Скрипт для интервью с тестерами

После того как человек попробовал приложение (15 мин звонок):

1. **Что было непонятно при первом использовании?**
2. **Что понравилось больше всего?**
3. **Стал бы использовать для реальных денег? Почему да/нет?**
4. **Что бы добавил в первую очередь?**
5. **Кому бы порекомендовал это приложение?**
